# Wayfield Labs

**The field of ways for community AI.**

Wayfield is a free, MIT-licensed framework for building AI that lives with your community — not in a corporate cloud. It runs entirely in the browser on free tiers, using WebLLM and Transformers.js.

We build tools that are together, findable, and shareable.

### The three ways

- **Kune** · *together* — Your AI that learns with you. Local chat, memory, and summarization that runs in the browser.
- **Trovi** · *find* — Privacy search for your archives. In-browser embeddings (all-MiniLM-L6-v2) with zero data leaving the device.
- **Havi** · *share* — Image and file hosting for communities, built for GitHub Pages and free tiers.

### Why Wayfield

1. **Free tier first** — Designed to run on Vercel Hobby, Cloudflare Pages, and GitHub Pages.
2. **Local by default** — WebGPU inference, IndexedDB storage, no API keys required.
3. **Community owned** — MIT license, open roadmap, built in public.
